# EliteJet Frontend

## Overview

EliteJet is a luxury private jet booking platform developed using React. The frontend provides a modern user interface for clients and administrators to manage aircraft bookings, view fleet information, and monitor booking status.

## Features

### Client Features

* User Registration
* User Login
* JWT Authentication
* Browse Available Aircraft
* View Aircraft Details
* Submit Booking Requests
* View Personal Bookings
* View Flight Ticket After Confirmation

### Admin Features

* Admin Dashboard
* Manage Aircraft Fleet
* Manage Bookings
* Manage Users
* Update Booking Status
* Delete Bookings and Users

### Additional Features

* Weather API Integration
* Protected Routes
* Role-Based Navigation
* Responsive Design

## Technologies Used

* React
* React Router DOM
* Axios
* CSS
* Vite

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
